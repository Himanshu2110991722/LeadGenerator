📖 API Setup Guide
How to get your free API keys for each source
OpenStreetMap — 100% Free, No Key Needed
Uses the Overpass API — completely free, unlimited, no signup required. Data comes from community contributors so phone numbers and websites may be missing for some businesses.
  1. No setup needed — just select OSM and start scraping!
  2. Best for: addresses, business names, basic info
  3. Tip: combine with Outscraper for phone/website data
Outscraper — 25 Free Searches/Month
Outscraper scrapes Google Maps directly. Each search = 1 request. Free tier gives 25 searches/month. Best quality data with phone numbers and websites.
  1. Go to outscraper.com → Sign up free
  2. Dashboard → API Keys → Create new key
  3. Copy the key and paste it in LeadHunterPro
  4. Free: 25 requests/month | Paid: from $3
Apify — $5 Free Credit on Signup
Apify's Google Maps Scraper actor is one of the best. You get $5 free credit on signup which covers hundreds of results. Uses Actor token for auth.
  1. Go to apify.com → Sign up free (get $5 credit)
  2. Settings → Integrations → API Tokens → Create token
  3. Copy the token and paste it in LeadHunterPro
  4. Actor used: compass/crawler-google-places
